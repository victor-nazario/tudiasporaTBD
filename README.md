# README #

In order to run the application, the user needs to open the directory created for the project, where its bin and src files are located at. To execute the app the user should run the following commands, each with a brief description. 


### Executable commands for Part1Main###


	 $ java -classpath ./bin  p1MainClasses/Part1Main 1
	 $ java -classpath ./bin  p1MainClasses/Part1Main 2
	 $ java -classpath ./bin  p1MainClasses/Part1Main 3
	 $ java -classpath ./bin  p1MainClasses/Part1Main 4
	 $ java -classpath ./bin  p1MainClasses/Part1Main 

### Executable commands for Part2Main###


	 $ java -classpath ./bin  p1MainClasses/Part2Main  n  m  isize  fsize  istep  rep
	 
	 
Where the variable values represent the following data: 

* n - the number of companies	
* m - the number of crime events
* isize - the initial size for experimentations
* fsize - final size for experimentations
* istep - increment of sizes
* rep - number of repetitions for a each size


### Command descriptions ###

*  Part1Main1-4: Provides the result generated by each independet solution
*  Part1Main : Provides the result generated by all solutions
*  Part2Main : Provides empircal results ofthe executon times of each indidual implementation for the provided user values
